巴菲特的裁决

"这不是一门好生意，但这是一个好价格。" 国银金租本质上是一家"加了10倍杠杆的银行"——用3.5%的成本借钱、以4.5%出租飞机船舶，赚1%的利差。ROE 11.7%听起来体面，但全靠高杠杆撑起。巴菲特一生回避这种"肥尾风险"生意——利率一动，利差就塌。

然而，市场给了它一个格雷厄姆式的烟蒂价格：P/B 0.43x、P/E 3.6x、股息率6.9%，意味着即使公司不再增长、甚至资产打对折，买入价也有安全边际。国开行控股64.4%的"隐性政府信用担保"+拨备覆盖率551%+不良率0.56%，让这张"负债表"比大多数中国银行都干净。这不是巴菲特的"美好生意"，是格雷厄姆的"肥雪茄还能抽三口"——适合收息党与深度价值玩家，不适合成长股信徒。

巴菲特测试：他们是主人翁型经理人吗？

国银金租的管理层是典型的"中国央企职业经理人"——能力合格、风控严谨、按部就班，但不是杨元庆/马化腾那种与股东命运绑定的创始人。高管薪酬按央企标准（一般百万级人民币），持股几乎为零，不是"主人翁"。

但另一面，国企背景也带来了极其保守的资产负债表经营——不良率0.56%、拨备覆盖551%，在中国所有金租公司中属于第一梯队。他们不会像爱尔兰AerCap那样激进扩张，但也不会爆雷。

收入模型的本质：利差生意

资金成本: ~3.5% (国开行批发资金 + 绿色金融债) 租赁收益率: ~5.0-5.5% (飞机/船舶) / ~4.0-4.5% (基础设施) ━━━━━━━━━━━━━━━━━━━━━━━━ 净利差: ~1.5-2.0% (这就是全部生意) 关键公式: 净利润 ≈ 生息资产 × 净利差 + 其他收入 - 拨备 - 运营成本 ≈ RMB 3,500亿 × 1.8% - RMB 20-30亿 拨备 - RMB 20亿 费用 = RMB 50亿
巴菲特的担忧：杠杆10倍+、利差0.5%的波动 = 净利润30%的波动
如果中国十年期国债利率下行导致资产端收益率先于负债端下降，利差可能从1.8%压缩到1.5%——净利润将腰斩。这就是为什么巴菲特说"杠杆让好生意变差，让差生意变烂"。

经济护城河分析

1. 网络效应无
2. 转换成本 (租约期内客户锁定)窄
3. 成本优势 (国开行批发资金)窄
4. 无形资产 (金融租赁牌照+政府信用)窄
5. 有效规模无
总体护城河评级：窄护城河

国银金租的核心优势是"国开行亲儿子"的资金成本优势——它能以3.5%借到10年期资金，而民营金租公司要5-6%。这个2%的利差优势就是它的护城河。

但这条护城河有两个致命弱点：(1) 政策可撤销——一旦国开行战略调整或央企"退金令"落实，低成本资金可能消失；(2) 招行金租/工银金租等银行系竞对同样享有资金优势，护城河并非独占。

关键问题：10年后这条护城河会更宽还是更窄？
更窄的可能性更大。原因：(1) 中国利率市场化接近完成，银行间资金成本差距在缩小；(2) 外资飞机租赁公司（AerCap、SMBC Aviation）在中国放开后会加剧竞争；(3) 金租行业需求增速放缓（中国"去杠杆"基调），行业Alpha收窄。

杜邦分析：ROE的拆解

ROE 11.7% = 净利率 17.8% × 资产周转 0.068x × 权益乘数 ~9.7x = (1.17% ROA) × (9.7x 杠杆) 对比银行业均值: ROE 10% = ROA 0.9% × 杠杆 11x 对比AerCap: ROE 13-18% = ROA 2.5% × 杠杆 5.5x

国银金租的ROA 1.17%比中国银行业平均（0.9%）好，比AerCap（2.5%）差。主要差距在飞机租赁资产的定价能力——AerCap有全球化客户+新老机队组合优势，国银金租仍以国内航司为主。

分红率回落的启示
2024年分红率从33%降至25%，公司明确说明"加强利润留存以支持资本补充"——这既是保守做法（长期正确），也是监管压力（资本充足率）。2025年维持25%分红率，意味着公司优先保资本、股息率不会大幅提升。

现金流质量说明

金融租赁企业的"自由现金流"概念不直接适用——新增业务本身就是资本密集型投放。更合适的指标是：

  • 经营活动现金流：2024年 ~RMB 500-600亿（主要是租金回收）
  • 投资活动现金流：大幅净流出（新增飞机/船舶/基础设施投放）
  • 融资活动现金流：大幅净流入（发债+同业拆借）
  • 净利润现金质量：净利润与派息基本匹配，会计稳健，无应计利润异常

法2：DCF (现金流折现)

金租企业DCF不直接适用（"FCF"被新增投放吞噬），用"净利润折现法"代替：

假设: 未来10年净利润增速 5% → 永续4%, r = 10% 2026E净利润 = 50.3 × 1.05 = 52.8亿 2035E净利润 = 52.8 × 1.05^9 = 82亿 终值 = 82亿 / (10% - 4%) × 1/(1.10)^10 = 527亿 10年现金流现值 ≈ 350亿 企业价值 ≈ 880亿 RMB = HK$966亿 每股 = HK$966亿 / 126.4亿股 = HK$7.6 (过于乐观) 保守情景: 假设永续增长0%, r 11% 每股 ≈ HK$2.80

DCF合理区间 HK$2.50 - HK$3.50 (多情景)，中位数 HK$3.00

法3：DDM (股利折现模型)

D0 = HK$0.109 假设增长率 g = 4% (与业务一致) 股权成本 r = 9-10% Gordon Growth: P = D0×(1+g)/(r-g) r=10%, g=4%: P = 0.109×1.04/0.06 = HK$1.89 r=9%, g=4%: P = 0.113/0.05 = HK$2.27 r=9%, g=5%: P = 0.114/0.04 = HK$2.87

DDM合理区间 HK$1.89 - HK$2.87，中位数 HK$2.30（DDM偏低因分红率仅25%）

法4：所有者盈余收益率（巴菲特偏好）

净利润 50.3亿 ≈ 所有者盈余 (金租业务拨备已计提) 所有者盈余收益率 @HK$1.59 = 50.3亿 × 1.10 / HK$201亿 = 27.5% 对比美国10年期国债 4.5% → 超额收益 23%

这是一个极高的盈余收益率，远超巴菲特要求的"至少2倍长债收益率"门槛（9%）。暗示严重低估。合理价 HK$3.00+

法5：PEG比率

P/E = 3.6x 未来3年净利润CAGR 预期 = 8-10% PEG = 3.6 / 9 = 0.40 (极度便宜, <1为便宜)

法6：NAV (净资产价值)

BVPS HK$3.73。在极端清算情景下（飞机/船舶打8折变现，不良资产全损），清算价值约 HK$3.73 × 0.75 = HK$2.80（仍高于现价）。

法7：反向DCF

当前市值 HK$201亿隐含假设: - 假设 r = 10% - 解出永续增长率 g: HK$201亿 × (10% - g) / 50.3亿×1.10 = 1 → g = -0.3% 即市场假设公司永续零增长甚至衰退。这与公司2016-2025 CAGR +10%的历史完全不符。
